AniFan — Endterm Project

Project Description
AniFan is a website dedicated to anime, manga, and games.  
It was created as an extension of a midterm project and enhanced with interactive features built using JavaScript and jQuery to improve the user experience.

Implemented Features
- Feedback form with validation
  - Validation of required fields: name, email, password, password confirmation.
  - Email validity check.
  - Password match check.
  - Real-time feedback via a modal window.

- Modal window (Popup)
  - Used to display error messages and confirm successful form submission.
  - Controlled programmatically via jQuery (`.modal(“show”)`).

- Gallery with filtering
  - The image gallery is divided into categories: Anime, Manga, Games.
  - Filter buttons hide/show images with smooth transitions (`.fadeIn()`, `.fadeOut()`).
  - Implemented using jQuery `.filter()` and effects.
Technologies
- HTML5 — site structure.
- CSS3 + Bootstrap 5 — styling and responsiveness.
- JavaScript + jQuery — interactivity, validation, modal windows, filtering.

How to use
1. Open the site in your browser.
2. Navigation is available via the top menu (Anime, Manga, Games, Gallery, News, Contacts).
3. In the Gallery section, use the filter buttons to view images by category.
4. In the Contacts section, fill out the feedback form:
   - Enter your name, email, password, and password confirmation.
   - A modal window with a message will appear upon success or failure.
5. All styles and scripts are loaded automatically; no additional action is required.Author
